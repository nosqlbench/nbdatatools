# Dataset Layout Specifications

The documents in this directory describe the canonical NBDataTools dataset layout:

- [Dataset Layout v1](dataset_writers_v1.md) – defines the `dataset.yaml` manifest and required facets.
- [Dataset Reader Contract v1](dataset_readers_v1.md) – describes how readers should interpret manifests and facet files.
- [Metadata Schema v1](metadata_writers_v1.md) – lists required/optional attributes for dataset manifests.
- [Metadata Reader Guidance v1](metadata_readers_v1.md) – explains how consumers should validate metadata.
- [Attribute Syntax v1](attribute_syntax_v1.md) – documents the canonical attribute specifier format used in manifests and tooling.
- [Predicate Format v1](predicate_format_v1.md) – describes the JSON predicate specification used in dataset manifests.
- [Ordinal Predicate Writers v1](ordinal_predicate_writers_v1.md) – gives guidance on writing predicate files for manifests.
